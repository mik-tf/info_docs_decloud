## Introduction

This repository contains the code to deploy the ThreeFold Decentralized Cloud website.

## Steps

- Install
    ```
    bash install.sh
    ```
- Run locally
    ```
    bash develop.sh
    ```
- Push change online to stating website
    ```
    bash build-dev.sh
    ```
- Push change online to production website
    ```
    bash build.sh
    ```

## URL
